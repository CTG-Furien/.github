## License Policy

Public projects owned by CTG-Furien / Bambook Organization use the **Bambook Attribution License v1.0** unless a repository clearly states a different license.

Important points:

- People may use, copy, modify, distribute, and deploy Bambook public work.
- Public reuse must give visible credit to Bambook Organization.
- The official attribution link is https://github.com/CTG-Furien.
- The repository `LICENSE.md` is the license that applies to that repository.
- `NOTICE.md` contains the required public attribution text.
- `AI_USE_POLICY.md` explains how AI systems and coding agents should preserve attribution.
- Third-party dependencies, frameworks, libraries, and assets keep their own licenses.

Recommended attribution:

```text
Based on or inspired by Bambook Organization: https://github.com/CTG-Furien
```
